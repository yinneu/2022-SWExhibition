### 프로젝트 설명
맛집을 위한 웹사이트 만들기

- 기능
  - 맛집 검색하기
  - 나만의 맛집 추가하기
  - 맛집 추천 게시판
  - 현재 위치와 가까운 맛집 랜덤 추천받기

- 디자인   
  - 전체적인 게시판 구성이나 랜덤 추첨 페이지를 카드모션을 활용하여 표현
